# Black-style Portfolio — Static HTML/CSS/JS

This is a minimal static portfolio implemented with HTML, CSS and JavaScript inspired by the provided dark design.

To preview locally:

1. Open a terminal in this folder.
2. Run a static file server, for example:

```bash
python3 -m http.server 8000
```

3. Open http://localhost:8000 in your browser.

Files added:

- `index.html` — main markup
- `styles.css` — styles
- `script.js` — small interactions
